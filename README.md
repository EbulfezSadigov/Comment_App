# Comments App

## Overview
This is a simple application that allows users to add and delete comments. The app uses React, Redux, TypeScript, and Material-UI for a modern UI experience.

## Features
- Add comments
- Delete comments
- Persistent state using localStorage

## Technologies Used
- React
- Redux
- TypeScript
- Material-UI
- Jest (for testing)
